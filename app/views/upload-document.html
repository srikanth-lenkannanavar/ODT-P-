How to save your email as a file which can be uploaded to your booking

PC
  • Open email
  • Click File and then select Print
  • Change printer to Microsoft XPS Document Writer and click print
  • Chose file name and location
  • Click Save
  • Click Upload on onthedot.co.uk and attach the file
 Mac
  • Open email
  • Click File and select Print
  • Select Save as PDF
  • Click Upload on onthedot.co.uk and attach the file
 Blackberry
  • Open email on screen
  • Take a screen capture by pressing the Volume Up key and Volume Down key at the same time. Screen captures are automatically saved in the Main pictures / Camera folder. You are only able to upload one file against each order, so please make sure you include all of the relevant information in a single screen capture
  • Click Upload on onthedot.co.uk and attach the file
 Android
  • Open email
  • In Menu options, select Print
  • Select Save as PDF. Android handsets vary so if you are unable to save your email as a PDF please take a screen capture
  • Click Upload on onthedot.co.uk and attach the file
 IPhone / IPad IOS
  • Open email on screen
  • Take a screen capture by simultaneously pressing the Home and power buttons. Screen captures are automatically saved in the Main photos / Camera roll folder. You are only able to upload one file against each order, so please make sure you include all of the relevant information in a single screen capture
  • Click Upload on onthedot.co.uk and attach the file
